# 🍞 Nomad Ledger (ノマド・レジャー)

Nomad Ledger は、小規模店舗やマルシェ、キッチンカーなどの出店に最適な、シンプルで高機能なモバイルレジシステムです。
Google スプレッドシートをデータベースとして活用し、一日の終わりには **Gemini AI が売上・時間帯トレンド・天気予報を分析して Slack に日報を届けてくれます。**

## 🚀 アプリのURL（ここから使えます）
**https://bokuranboraemon.github.io/nomad-ledger/**

※初回アクセス時に設定画面からGASのURLを登録してください。
---

## 🚀 導入ガイド (4ステップ)

初めて使う方は、以下の手順で自分専用の環境を作成してください。

### Step 1: スプレッドシートの準備
1. 配布された **スプレッドシートのコピー用URL**（作成者から共有されたもの）を開きます。
2. https://docs.google.com/spreadsheets/d/1a8ZdX4jVOGS2kuCk56G_9ZWaLSTi9H5q0lFKzibuOEc/copy
3. 「コピーを作成」をクリックして、自分の Google ドライブに保存します。
4. シート内の **「商品マスタ」** タブに、あなたのお店の「商品名」と「価格」を入力してください。

### Step 2: 接続URL (API) の発行
ここが最も重要なステップです。
1. コピーしたスプレッドシートのメニューから **「拡張機能」＞「Apps Script」** を開きます。
2. 画面右上の青い **「デプロイ」** ボタンをクリックし、 **「新しいデプロイ」** を選びます。
3. 種類の選択（歯車アイコン）から **「ウェブアプリ」** を選択します。
4. 以下の設定を行い、「デプロイ」をクリックします。
   - **説明**: 任意（例: Nomad Ledger API）
   - **実行ユーザー**: 自分
   - **アクセスできるユーザー**: **全員**
5. Google から承認を求められたら許可してください。
   - ※「このアプリは確認されていません」と出た場合は、**「詳細」→「Nomad Ledger（安全ではないページ）に移動」** をクリックして許可を進めてください。
6. 最後に表示された **「ウェブアプリURL」** (末尾が `/exec` のもの) をコピーしてメモしておきます。

### Step 3: 外部APIとSlack連携の設定 (AI日報用)
AIによる高度な売上分析と天気予報連携を機能させるため、各種APIキーを取得してスプレッドシートに登録します。

**【各キーの取得方法】**
* **Gemini APIキー**:
    1. [Google AI Studio](https://aistudio.google.com/) にアクセスし、Googleアカウントでログインします。
    2. 左側のメニューから「Get API key」をクリックし、「Create API key」から新しいキーを生成してコピーします。
* **OpenWeather APIキー**:
    1. [OpenWeatherMap](https://openweathermap.org/) にアクセスし、無料アカウントを作成してログインします。
    2. 右上のアカウント名をクリックし、「My API keys」からキーをコピーします。（※作成直後はキーが有効になるまで数時間かかる場合があります）
* **Slack Webhook URL**:
    1. 日報を受け取りたいSlackのワークスペース設定を開き、「Incoming Webhooks」アプリを検索して追加します。
    2. 通知を送りたいチャンネルを指定し、発行された `https://hooks.slack.com/services/...` から始まるURLをコピーします。

**【キーの登録手順】**
1. スプレッドシートの画面に戻り、ブラウザの更新ボタン（F5）でページを再読み込みします。
2. 数秒待つと上部メニューに **「⚙️ Nomad Ledger管理」** が追加されるので、**「🔑 APIキー・Webhookの登録」** をクリックします。
3. ポップアップの指示に従って、取得した3つの情報を順番に貼り付けて保存してください。（ソースコード上には見えない、安全な領域に保存されます）

### Step 4: アプリの設定
1. アプリのURL（ `https://bokuranboraemon.github.io/nomad-ledger/` ）を端末で開きます。
2. 右上の **「⚙️ 設定」** アイコンをタップします。
3. 「GAS エンドポイントURL」の欄に、Step 2 でコピーしたURLを貼り付けて **「保存」** を押します。
4. 設定画面の **「☁️ スプレッドシートから商品リストを最新化」** を押すと、あなたの商品リストが読み込まれます！

## 📱 便利な使い方
### ホーム画面に追加して「アプリ化」する
- **iPhone/iPad**: Safari で開き、共有ボタン ＞ 「ホーム画面に追加」
- **Android**: Chrome で開き、三点ドットメニュー ＞ 「ホーム画面に追加」
これで次回から、全画面のレジアプリとして起動できます。

### 売上の確認と AI 日報
- 売上データはスプレッドシートに月次タブ（例：売上履歴_2026-04）で自動保存されます。
- GASの画面（拡張機能 ＞ Apps Script）の左側メニューから時計アイコン（トリガー）を開き、`generateDailyReport` 関数を「時間主導型」「日付ベースのタイマー」で毎日閉店後の時間（例：午後6時～7時）に実行されるよう設定しておくと、AI日報がSlackへ自動送信されます。

## 🛠️ システム構成
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla JS)
- **Backend**: Google Apps Script (GAS)
- **AI**: Gemini 2.0 Flash (via Gemini API)
- **Storage**: LocalStorage (Offline support) & Google Sheets
